\mainpage

计算几何算法库(The Computational Geometry Algorithms Library, \cgal)是一个软件项目，以 C++ 库的形式提供高效可靠的几何算法实现。

<h2>功能包概览</h2>

\cgal 提供了丰富的数据结构和算法,包括 \ref PartTriangulationsAndDelaunayTriangulations "三角剖分(triangulations)"、
\ref PartVoronoiDiagrams "Voronoi图"、\ref PartPolygons "多边形"、\ref PartPolyhedra "多面体"、
\ref PartArrangements "曲线排列(arrangements)"、\ref PartMeshing "网格生成"、
\ref PartGeometryProcessing "几何处理"、\ref PartConvexHullAlgorithms "凸包算法"等。

这些数据结构和算法都是基于点、线段等几何对象进行操作,并对它们执行几何测试。
这些对象和谓词都被组织在 \ref PartKernels "CGAL 几何核(Kernels)"中。

此外,\ref PartSupportLibrary "支持库(Support Library)"提供了几何对象生成器、空间排序函数,
以及矩阵搜索框架和线性/二次规划求解器。它还提供了与第三方软件的接口,如 Qt 图形界面库
和 Boost Graph Library。

完整的功能包列表可以在 \ref packages 页面查看。

<h2>快速入门</h2>

请访问 \ref general_intro 了解如何获取、安装和使用 \cgal。

<h2>许可协议</h2>

\cgal 采用双重许可协议。在开源软件中使用 \cgal 是免费的。如需在其他场景下使用 \cgal,
可以从 [GeometryFactory](https://www.geometryfactory.com) 获取商业许可。
详细信息请参见 \ref license "许可协议"页面。

<h2>致谢</h2>

详见 \ref how_to_cite_cgal 页面。
